# opengl-haven
A project based around learning OpenGL. This repo will contain files about different rendering techniques and scenarios in OpenGL.

What will be demonstrated [✅/❌]:
- Making a Window [❌]
- Storing vertices [❌]
- Compiling shaders [❌]
- Drawing a triangle [❌]
- Loading Textures [❌]
- Applying Transformations [❌]
- Loading Object models [❌]
